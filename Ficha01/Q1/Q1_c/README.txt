O código presente nessa pasta necessita de uma organização específica de entrada

A entrada consiste em 3 números separados por espaço e com o sinal identificado 
SOMENTE por "+" para número spositivos e "-" para números negativos.

Ex1:
+125 -5 +24 ;

Não há espaços entre os números e seus respectivos sinais. É NECESSARIO que a entrada 
termine com algum outro caracter que não seja número ou algum dos sinais pre-citados.
Um espaço é suficiente.
